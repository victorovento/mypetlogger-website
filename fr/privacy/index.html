Politique de confidentialité

Dernière mise à jour : 20 avril 2026 · Español

Version courte : MyPetLogger est un journal pour animaux privé, sur l'appareil. Nous n'avons pas de comptes, nous n'avons pas de serveurs à nous, et nous ne collectons aucune donnée personnelle sur vous ou vos animaux. Vos enregistrements vivent sur votre iPhone, et se synchronisent éventuellement sur vos propres appareils via votre propre iCloud. Ni nous ni aucun tiers ne pouvons les lire.

1. Qui nous sommes

MyPetLogger est développée et publiée par LCVSystems LLC (« LCVSystems », « nous »), une société à responsabilité limitée des États-Unis. Vous pouvez nous joindre à support@mypetlogger.com.

2. Quelles données nous collectons

MyPetLogger est une application hors ligne d'abord. Les données principales sur les animaux que vous saisissez — noms, espèces, races, dossiers médicaux, promenades, notes, photos — sont stockées localement sur votre appareil dans le framework Core Data d'Apple, et ne nous sont jamais transmises.

Les catégories de données limitées suivantes peuvent être collectées par des services tiers :

  • Apple (synchronisation iCloud) : si vous activez la synchronisation iCloud dans les Réglages, vos données sur les animaux sont répliquées vers votre propre base de données iCloud privée à l'aide de CloudKit d'Apple. LCVSystems n'a aucun accès à ces données.
  • Apple (analyses de plantage) : si vous avez accepté de « Partager avec les développeurs d'applications » dans les Réglages d'iOS, Apple peut nous transmettre des journaux de plantage anonymes. Ceux-ci ne contiennent aucune donnée personnelle.
  • Apple HealthKit : si vous activez « Synchroniser les promenades avec Apple Santé », les séances de promenade sont écrites dans votre propre base de données Santé sur l'appareil. LCVSystems n'y a pas accès.
  • Google AdMob (utilisateurs gratuits uniquement) : le SDK AdMob peut collecter des identifiants au niveau de l'appareil et des métriques agrégées pour diffuser et mesurer les publicités. AdMob est régi par la politique de confidentialité de Google. Vous pouvez vous désinscrire en vous abonnant à un niveau payant. Au premier lancement, nous demandons l'autorisation App Tracking Transparency ; si vous refusez, seules des publicités non personnalisées sont diffusées.
  • Google Mobile Ads User Messaging Platform (UMP) : pour la gestion du consentement dans les régions applicables.
  • Google Analytics (site Web uniquement) : ce site Web marketing utilise Google Analytics avec l'anonymisation IP activée pour mesurer les visites agrégées. L'application elle-même n'utilise pas Google Analytics.

3. Ce que nous NE faisons PAS

  • Nous n'exigeons, ne créons, ni ne stockons de comptes utilisateur.
  • Nous n'exploitons aucun serveur dorsal qui stocke vos données sur les animaux.
  • Nous ne vendons, ne louons, ni ne partageons vos données personnelles avec des tiers à des fins de marketing.
  • Nous n'utilisons pas vos données pour entraîner des modèles d'apprentissage automatique.
  • Nous ne construisons pas de profils publicitaires basés sur votre activité dans l'application.

4. Autorisations que nous demandons

  • Appareil photo et photothèque : pour prendre ou choisir une photo d'animal.
  • Localisation (lors de l'utilisation et toujours) : pour enregistrer les itinéraires de promenade. « Toujours » n'est demandée que pour continuer à suivre votre promenade lorsque l'application est en arrière-plan.
  • HealthKit : pour écrire les promenades terminées comme séances dans Apple Santé. Demandé uniquement la première fois que vous commencez une promenade, et seulement si vous l'activez.
  • Notifications : pour vous avertir lorsqu'un objectif de promenade est atteint ou qu'un élément de santé arrive à échéance.
  • App Tracking Transparency : requis par Apple avant qu'AdMob puisse utiliser votre identifiant publicitaire pour des publicités personnalisées.

5. Enfants

MyPetLogger n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons sciemment aucune information personnelle auprès d'enfants.

6. Vos droits

Parce que vos données vivent sur votre appareil, vous les contrôlez directement :

  • Supprimez n'importe quel enregistrement dans l'application, ou désinstallez l'application pour tout supprimer d'un coup.
  • Utilisez Réglages → Synchronisation iCloud → « Supprimer les données du cloud et désactiver la synchronisation » pour supprimer la copie répliquée par CloudKit de vos données de votre iCloud.
  • Contactez support@mypetlogger.com pour toute question, demande RGPD/CCPA, ou plainte.

7. Modifications

Nous pouvons mettre à jour cette politique. Les modifications importantes seront notées dans l'écran Réglages → Quoi de neuf de l'application et sur cette page. La poursuite de l'utilisation après une mise à jour constitue une acceptation.
